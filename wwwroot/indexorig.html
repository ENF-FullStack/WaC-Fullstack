Welcome to course

You now should have your node-based web-server running and this page is displayed at http:/localhost:9000.

If not, open command prompt and cd into server directory. Then give command

>node basicserver.js

Most of the work you'll do in your wwwroot-directory; it contains the exercise templates.

The first exercises will be done with start.html. Later you will add styles to wwwroot/styles/start.css -file.

wwwroot/images-folder contains some images for your convenience.
