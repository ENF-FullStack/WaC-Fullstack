Form basics

Step 1, basic form

You already should have your node-based web server running. Follow your instructors lead to make it accept form-data at url "/formdata"

Start by single input-field with name attribute set to something and a submit button to test the form.

Experiment with get and post methods

Step 2, more form fields

Add some components to the form

  • Input field for entering birthday
  • Input for email
  • Listbox (select-field) displaying few items which can be purchased
  • Radiobuttons for selecting item type (downloadable, hardcopy, or something like)
  • Input field for quantity of items
  • Text area for entering extra information
  • Combobox (select-field) for selecting payment method (Money transfer, credit card)
  • Checkbox "I agree to terms"

Step 3, Finalizing

Specify labels for the controls

Use the autofocus, required and placeholder attributes

Step 4, Extra

Experiment with other input types. Try the datalist-feature. Remember to test with different browsers
